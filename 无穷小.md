**定义** 若 $\lim_{x\to x_0}f(x)=0$（或 $\lim_{x\to\infty}f(x)=0$），那么称函数 $f(x)$ 为当 $x\to x_0$（或 $x\to\infty$）时的无穷小。特别地，若 $\lim_{n\to\infty}x_n=0$，那么称数列 $\{x_n\}$ 为当 $n\to\infty$ 时的无穷小。

**注**：
①无穷小与 $x$ 的变化过程有关，$\lim_{x\to 0}\sin x=0$，可以说当 $x\to 0$ 时，$\sin x$ 为无穷小；而 $\lim_{x\to \frac{\pi}{2}}\sin x=1\ne 0$，所以 $x\to \frac{\pi}{2}$ 时，$\sin x$ 就不是无穷小了。
②很小的常数如 $0.00001$ 不是无穷小，但常数 $0$ 是无穷小。

**定理**（利用无穷小去极限号）
$\lim f(x)=A \Leftrightarrow f(x)=A+\alpha$，其中 $\lim \alpha=0$。这个定理可以用于去掉极限符号，且去掉后得到一个 $f(x)$ 的等式。

**无穷小的性质**
(1) 常数与无穷小之积也是无穷小。
(2) 有界函数与无穷小之积仍为无穷小（即有界函数与无穷小之积的极限为0）。
(3) 有限个无穷小之和也是无穷小，但无限个无穷小之和未必是无穷小。
(4) 有限个无穷小之积也是无穷小，但无限个无穷小之积未必是无穷小。


## 无穷小的比较

**无穷小的阶** 设 $\lim f(x)=0$ 与 $\lim g(x)=0$，但两者趋于 0 的速度有所不同，可通过计算极限 $\lim \dfrac{f(x)}{g(x)}$ 来比较两者趋于 0 的速度，这就是无穷小的比阶问题，比较结果常有如下 3 种情形：

(1) 如果 $\lim \dfrac{f(x)}{g(x)}=0$，则称 $f(x)$ 是比 $g(x)$ ==高阶无穷小==，记 $f(x)=o[g(x)]$。

(2) 如果 $\lim \dfrac{f(x)}{g(x)}=C\ne 0$，则称 $f(x)$ 与 $g(x)$ 是==同阶无穷小==。特别地，如果 $\lim \dfrac{f(x)}{g(x)}=1$，则称 $f(x)$ 与 $g(x)$ 是==等价无穷小==。记作$f(x) \sim g(x)$
另外，如果 $f(x)$ 与 $g^k(x)$，$k>0$ 是同阶的无穷小，即 $\lim \dfrac{f(x)}{g^k(x)}=C\ne 0$，则称 $f(x)$ 是 $g(x)$ 的 $k$ 阶无穷小。

(3) 如果 $\lim \dfrac{f(x)}{g(x)}=\infty$，则称 $f(x)$ 是比 $g(x)$ ==低阶无穷小==。 

**注**：
①在 $x\to 0$ 时，$n$ 越大，$x^n$ 越是更高阶的无穷小，$n$ 越小，$x^n$ 越是低阶无穷小。
②常数 $0$ 是一切无穷小量（$\ne 0$）的高阶无穷小。
③ $x\to 0$ 时，$o(x^n)$ 表示的是一类趋于 0 速度远快于 $x^n$ 函数，而不是一个具体函数。
④不同阶无穷小相加减具有舍高性：$\alpha\pm o(\alpha)\sim \alpha$，其中 $\lim \alpha=0$。

**高阶无穷小的运算性质**：
(1)合一性： $o(x^n)\pm o(x^n)=o(x^n)$；
(2)吸收性： $ko(x^n)=o(kx^n)=o(x^n)$；k非0
(3)舍高性： $o(x^m)\pm o(x^n)=o(x^m)\ (m<n)$；
(4)升阶性： $o(x^m)\cdot o(x^n)=o(x^{m+n})$；
(5)升阶性： $x^m\cdot o(x^n)=o(x^{m+n})$。
