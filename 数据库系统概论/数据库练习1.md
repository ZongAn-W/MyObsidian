## 练习题 1：图书馆借阅管理

已知如下业务背景：

```
读者(读者号, 姓名, 性别, 年龄)
图书(图书号, 书名, 作者, 馆藏地点)
借阅(读者号, 图书号, 借阅日期)
```

请写出完成下列任务的结构化查询语言语句：

1. 写出三张表的建表语句，并明确定义主键和外键。
2. 创建两个视图：
   - `图书借阅详情视图`：显示读者号、读者姓名、图书号、书名、借阅日期。
   - `图书馆藏视图`：显示图书号、书名、作者、馆藏地点。
3. 查询借阅了 `数据库系统` 或 `操作系统` 的读者号。
4. 查询没有借阅任何图书的读者全部信息。
5. 查询借阅了全部图书的读者全部信息。
6. 查询至少借阅了一本 `1001` 号读者借阅过的图书的读者号。
7. 查询每本图书被多少名读者借阅。

### 作答区

```
1.
CREATE TABLE 读者(
	读者号 VARCHAR(20) PRIMARY KEY,
	姓名 VARCHAR(50) NOT NULL,
	性别 CHAR(1),
	年龄 INT
)；
CREATE TABLE 图书(
	图书号 VARCHAR(20) PRIMARY KEY,
	书名 VARCHAR(50) NOT NULL,
	作者 VARCHAR(50),
	馆藏地点 VARCHAR(200)
)；
CREATE TABLE 借阅(
	读者号 VARCHAR(20),
	图书号 VARCHAR(20),
	借阅日期 DATE，
	PRIMARY KEY (读者号，图书号)，
	FOREIGN KEY (读者号) REFERENCES 读者(读者号)，
	FOREIGN KEY (图书号) REFERENCES 图书(图书号)
)；

2.
CREATE VIEW 图书借阅详情视图 (读者号，读者姓名，图书号，书名，借阅日期)AS
SELECT 读者.读者号，读者.姓名，借阅.图书号，图书.书名，借阅.借阅日期
FROM 读者，图书，借阅
WHERE 借阅.读者号=读者.读者号 AND 借阅.图书号=图书.图书号

CREATE VIEW 图书馆藏视图(图书号，书名，作者，馆藏地点)AS
SELECT 图书.图书号，图书.书名，图书.作者，图书.馆藏地点
FROM 图书
	
3.
SELECT 借阅.读者号
FROM 借阅，图书
WHERE 	借阅.图书号=图书.图书号AND 图书.书名 IN ("操作系统"，"数据库原理")

4.
SELECT *
FROM 读者
WHERE NOT EXISTS(
	SELECT *
	FROM 借阅
	WHERE 借阅.读者号=读者.读者号
)

5.
SELECT *
FROM 读者
WHERE NOT EXISTS(
	SELECT *
	FROM 图书
	WHERE NOT EXISTS(
		SELECT*
		FEOM 借阅
		WHERE 借阅.图书号=图书.图书号
		AND 借阅.读者号=读者.读者号
	)
)；

6.
SELECT 读者.读者号
FROM 读者，借阅
WHERE 借阅.图书号 IN(
	SELECT 借阅.图书号
	FROM 借阅，图书
	WHERE 借阅.读者号 = "1001"
	AND 图书.图书号=借阅.图书号
)AND 读者.读者号=借阅.读者号

7
SELECT 借阅.图书号 COUNT(读者号)AS借阅人数
FROM 借阅
GROUP BY 借阅.图书号

```

## 练习题 2：学生社团活动管理

已知如下业务背景：

```
学生(学号, 姓名, 性别, 年龄)
活动(活动号, 活动名, 负责人, 地点)
报名(学号, 活动号, 报名日期)
```

请写出完成下列任务的结构化查询语言语句：

1. 写出三张表的建表语句，并明确定义主键和外键。提示：`负责人` 参照 `学生(学号)`。
2. 创建两个视图：
   - `活动负责人详情视图`：显示活动号、活动名、负责人姓名、负责人性别。
   - `报名详情视图`：显示学号、学生姓名、活动号、活动名、报名日期。
3. 查询报名了 `篮球赛` 或 `歌唱比赛` 的学生学号。
4. 查询没有报名任何活动的学生全部信息。
5. 查询报名了全部活动的学生全部信息。
6. 查询至少报名了一个 `2023001` 号学生报名过的活动的学生学号。
7. 查询每个活动有多少名学生报名。

### 作答区

```
CREATE TABLE STU(
	SNO CHAR(20) PRIMARY KEY,
	SNA VARCHAR(50) NOT NULL,
	SEX CHAR(1),

);

CREATE TABLE ACT(

	ACTNA VARCHAR(50) NOT NULL,
	ACTRES VARCHAR(50),
	SITE VARCHAR(100),
	FOREIGN KEY (ACTRES) REFERENCES STU(SNA)

CREATE TABLE BM(

	ACTDA DATE,
	PRIMARY KEY (SNO，ACTNO)，
	FOREIGN KEY (SNO) REFERENCES STU(SNO)，
	FOREIGN KEY (ACTNO) REFERENCES ACT(ACTNO)
)；

2.
CREATE VIEW ACTVIEW(ACTNO，ACTNA，RESNO，RESSEX)AS
```
