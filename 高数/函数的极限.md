==邻域与去心邻域== 实数集 $\{x\mid |x-x_0|<\delta\}$ 即开区间 $(x_0-\delta,x_0+\delta)$ 称为点 $x_0$ 的 $\delta$ 邻域，记作 $U(x_0,\delta)$，$x_0$ 称为邻域中心，$\delta$ 称为邻域半径。

如果在 $U(x_0,\delta)$ 中去掉中心点 $x=x_0$，即实数集 $\{x\mid 0<|x-x_0|<\delta\}$ 或 $(x_0-\delta,x_0)\cup(x_0,x_0+\delta)$ 为点 $x_0$ 的 $\delta$ 去心邻域，记为 $\mathring{U}(x_0,\delta)$。


## $x\to x_0$ 时==函数的极限==

数学定义：$\forall \varepsilon>0,\exists \delta>0$，当 $0<|x-x_0|<\delta$ 时，有 $|f(x)-a|<\varepsilon$，则 $\lim_{x\to x_0}f(x)=a$。

$x\to x_0$ 包括了两个方向，$x\to x_0^-$ 和 $x\to x_0^+$。如果 $\lim_{x\to x_0^-}f(x)=A_1$，则 $A_1$ 称为 $f(x)$ 在 $x_0$ 处的左极限，记作 $f(x_0^-)=A_1$；
如果 $\lim_{x\to x_0^+}f(x)=A_2$，则 $A_2$ 称为 $f(x)$ 在 $x_0$ 处的右极限，记作 $f(x_0^+)=A_2$。

注：
①==双侧性== $\lim_{x\to x_0}f(x)$ 存在的充要条件是 $f(x_0^-)$ 与 $f(x_0^+)$ 都存在且相等。
②==去心性== $\lim_{x\to x_0}f(x)=a$ 与 $f(x_0)$ 是否等于 $a$ 无关。

## $x\to\infty$ 时函数的极限

数学定义：$\forall \varepsilon>0,\exists X>0$，当 $|x|>X$ 时，有 $|f(x)-a|<\varepsilon$，则 $\lim_{x\to\infty}f(x)=a$。

注：
① $x\to\infty$ 包含 $x\to-\infty$ 和 $x\to+\infty$ 两个方向，$\lim_{x\to-\infty}f(x)$ 也可记作 $f(-\infty)$，$\lim_{x\to+\infty}f(x)$ 也可记作 $f(+\infty)$。
② $\lim_{x\to\infty}f(x)$ 存在的充要条件是 $\lim_{x\to-\infty}f(x)$ 与 $\lim_{x\to+\infty}f(x)$ 都存在且相等。

## 需要分别计算 $f(x_0^-)$ 与 $f(x_0^+)$ 及 $f(-\infty)$ 与 $f(+\infty)$ 的极限。
一般情况下，$f(x_0^-)$ 与 $f(x_0^+)$ 或 $f(-\infty)$ 与 $f(+\infty)$ 是相同的，是不需要分别计算的，但见到如下几种情况时，极限结果可能会不同，是需要分别计算的：

① 如果 $x_0$ 是分段函数的分段点，则计算 $\lim_{x\to x_0}f(x)$ 要分别计算 $\lim_{x\to x_0^-}f(x)$ 与 $\lim_{x\to x_0^+}f(x)$。

② 由于 $e^{-\infty}=0,e^{+\infty}=+\infty$，所以出现 $e^\infty$ 型极限要分别计算 $f(-\infty)$ 与 $f(+\infty)$。

③ 由于 $\arctan(-\infty)=-\dfrac{\pi}{2},\arctan(+\infty)=\dfrac{\pi}{2}$，所以出现 $\arctan(\infty)$ 要分别计算 $f(-\infty)$ 与 $f(+\infty)$。

④ 由于 $\operatorname{arccot}(-\infty)=\pi,\operatorname{arccot}(+\infty)=0$，所以出现 $\operatorname{arccot}(\infty)$ 要分别计算 $f(-\infty)$ 与 $f(+\infty)$。

⑤ 含绝对值的函数，往往需要讨论绝对值内的符号，去掉绝对值再计算极限。
