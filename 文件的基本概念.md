- 文件是用户访问计算机的基本数据单位
- 文件是指具有文件名的若干相关元素（记录）的集合

### 文件的定义：
#### 文件属性：
- 文件名称
- 文件标识符
- 文件类型
- 文件位置
- 文件大小
- 文件保护
- 文件时间

#### 文件的类型：
- 按用途分：系统文件；用户文件；库文件
- 按文件中数据的形式分类：源文件；目标文件；可执行文件
- 按照存取控制属性分类：可执行文件；只读文件；读写文件
- 按组织形式和处理方式分类；普通文件；目录文件；特殊文件

## ==文件系统模型==


|    文件系统模型     | 例子              |
| :-----------: | --------------- |
|    文件系统接口     | 命令接口、程序接口       |
| 对对象操纵和管理的软件集合 | 实现文件系统全部功能的软件集合 |
|     对象及属性     | 文件、目录、磁盘        |
- 操作系统中用于组织、管理、访问持久化数据的分层抽象架构
- 把物理存储细节屏蔽，给用户/应用提供统一、易用的文件/目录接口
- ==文件管理系统管理的对象==：文件管理系统的核心实体和元数据：文件、目录、磁盘
- ==对对象操纵和管理的软件集合==：实现文件系统全部功能的软件集合
- ==文件系统的接口==：提供给用户的交互界面：命令接口、程序接口
