## 位示图
- 利用二进制的一位来表示磁盘中每个盘块的使用情况
- 所有二进制位构成一个集合称为位示图
- 0为盘块空闲，1为盘块已分配

盘块分配的步骤：
- 按顺序扫描位示图，从中找出一个或一组为0的二进制位(i,j)；
- 将找到的(i,j)转换为与之相应的盘块号b
- 修改位示图，使map\[i]\[j]=1
盘块回收的步骤：
- 将要回收的盘块号b转换为位示图中相应的(i,j)
- 修改回收区在位示图中对应的二进制位，令map\[i]\[j]=0

二维数组大小m×n
已知位示图的二进制位(i,j)求盘块号b
四种情况：
- 盘块号从0计数，数组下标从0开始：$b=i×n+j$
- 盘块号从0计数，数组下标从1开始：$b = (i-1) \times n + (j-1)$
- 盘块号从1计数，数组下标从0开始：$b = i \times n + (j+1)$
- 盘块号从1计数，数组下标从1开始：$b = (i-1) \times n + j$

已知盘块号b求位示图的二进制位(i,j)
- 盘块号从0计数，数组下标从0开始：
行号$i = b \div n$（取整）
列号$j = b \mod n$（取余）
- 盘块号从0计数，数组下标从1开始：
行号$i = b \div n+1$（取整）
列号$j = b \mod n+1$（取余）
- 盘块号从1计数，数组下标从0开始：
行号$i = (b-1) \div n$（取整）
列号$j = (b-1) \mod n$（取余）
- 盘块号从1计数，数组下标从1开始：
行号$i = (b-1) \div n+1$（取整）
列号$j = (b-1) \mod n+1$（取余）
