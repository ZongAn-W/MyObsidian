
# 读文件
- 通过[[文件描述符FD]]发起读操作
- OS通过FD找到对应的打开文件表项
- 通过表项找到内存inode
- 按文件偏移量定位到外存数据块
- 将文件从外存读入内存缓冲区
- 传递给用户进程，更新文件偏移量
#### 步骤
- 通过FD定位句柄：内核根据进程的文件描述符，用FD作为索引，找到对应文件句柄
- 通过句柄获取inode：从文件句柄中取出inode指针，定位到该文件的inode结构体
- 通过inode读取数据：内核根据inode中的数据块指针，找到文件内容在磁盘上的物理存储位置，读取数据返回给进程

# 写文件
- 通过[[文件描述符FD]]发起读操作
- OS通过FD找到对应的打开文件表项
- 通过表项找到内存inode
- 按文件偏移量定位到外存数据块
- 将用户进程数据写入内存缓冲区，再同步到外存
- 更新文件偏移量和inode元信息，完成写操作
#### 步骤
- 通过FD定位句柄：内核根据进程的文件描述符，用FD作为索引，找到对应文件句柄
- 通过句柄获取inode：从文件句柄中取出inode指针，定位到该文件的inode结构体
- 通过inode写入数据：内核根据inode中的数据块指针，找到文件在磁盘上的物理存储位置，将进程数据写入对应磁盘数据块，更新inode相关信息，完成写操作
