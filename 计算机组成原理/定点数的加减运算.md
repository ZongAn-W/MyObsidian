补码的加减运算：
- 整数：(A+B)mod $2^{n-1}$=A+B
- 小数：(A+B)mod $2$=A+B

- 整数：(A-B)mod $2^{n-1}$=A+(-B)
- 整数：(A-B)mod $2$=A+(-B)

## 溢出和判断方式
- 上溢：正数+正数=负数
- 下溢：负数+负数=正数
- 正数+负数 不可能溢出

方法二：==采用一位符号位，根据数据位的进位情况判断==

|     | 符号位的进位 | 最高数位的进位 |
| --- | ------ | ------- |
| 上溢  | 0      | 1       |
| 下溢  | 1      | 0       |

方法三：采用==双符号位==
- 正数的符号位：00
- 负数的符号位：11
- 01：上溢
- 10：下溢
