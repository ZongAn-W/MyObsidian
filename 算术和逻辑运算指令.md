# add /sub指令
- add 将两个操作数相==加==结果保存在第一个操作数中：add eax,10//eax=eax+10
- sub 将第一个操作数==减==去第二个操作数，结果保存在第一个操作数中
# imul指令
- ==有符号整数乘法==指令，支持双操作数和三操作数
- 双操作数：将两个操作数相乘，结果保存在第一个操作数中
- 三操作数：将第二三个操作数相乘结果存在第一个操作数中
# mul指令
- ==无符号整数乘法==，仅支持单操作数格式，被乘数隐含在寄存器中
# idiv/div指令
- ==有/无符号整数除法指令==，仅仅指定除数，被除数为edx:eax组成的联合寄存器中。执行后，商存入eax，余数存入edx
- idiv ebx //eax=(edx:eax)/ebx，余数存入edx
# inc/dec指令
- inc将操作数==自增1==、dec将操作数==自减1==
# and/or/xor指令
- 分别按位 与、或、异或操作，结果存入第一个操作数
- 一般用来置位、清零操作
- and eax,0fH //将eax高位置零，低四位保持不变
- or bl，80H //将bl的最高位置1
- xor edx,edx // 将edx清零
# not指令
- 按位取反指令
# neg指令
- 取负指令
# shl/shr指令
- 逻辑左移/逻辑右移
- 第一个操作数是被移位数
- 第二个是移位的位数
- sal/sar 为算术移位指令
