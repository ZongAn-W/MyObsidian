# mov 指令
- 将第二个操作数（寄存器、内存、常数值）复制到第一个操作数（寄存器、内存）
格式：
- $mov <reg>,<reg>$
- $mov <reg>,<mem>$
- $mov <mem>,<reg>$
- $mov <reg>,<con>$
- $mov <mem>,<con>$
- [[常用指令]]
- 双操作数的两个操作数不能同时在内存，不能直接从内存复制到内存

# push 指令
- 将操作数压入栈中，常用于函数调用和现场保护，ESP是栈顶，入栈前先将栈顶减4（默认32位），再将操作数压入ESP所指位置
格式：
- $push <reg>$
- $push <mem>$
- $push <con>$

# pop 指令
- 从栈中弹出数据，出栈前先将ESP所指数据读出，再将ESP加4
- pop eax//弹出栈顶元素存入eax里

# lea 指令
- 计算地址并赋值，不读内存
- 计算源操作数的有效地址并传送到目的寄存器。
- lea eax,[ebx+ecx\*4+8]//计算[ebx+ecx\*4+8]地址并存入eax
